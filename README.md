# AgroSales

Plataforma de gestão comercial agrícola composta por:

- sistema Web interno em Angular;
- aplicativo Android em Kotlin;
- API em FastAPI;
- banco PostgreSQL;
- infraestrutura com Docker.

## Status

Projeto em desenvolvimento.
